List repositories or entities within a repository

Usage: nap list [OPTIONS] [REPOSITORY]

Arguments:
  [REPOSITORY]  Repository name. Omit to list all repositories

Options:
  -d, --base-dir <BASE_DIR>        Base directory for repository repositories. Defaults to $NAP_DIR, or ~/.nap if unset [env: NAP_DIR=]
  -t, --entity-type <ENTITY_TYPE>  Entity type to list (if repository is specified)
  -v, --verbose                    Enable verbose debug logging
  -h, --help                       Print help
