Create a new entity manifest

Usage: nap create [OPTIONS] --repository <REPOSITORY> --name <NAME> <ENTITY_TYPE> <ENTITY_ID>

Arguments:
  <ENTITY_TYPE>  Entity type (any non-empty string, e.g. character, location, custom-type)
  <ENTITY_ID>    Entity ID (slug). e.g., "lukeskywalker"

Options:
  -d, --base-dir <BASE_DIR>      Base directory for repository repositories. Defaults to $NAP_DIR, or ~/.nap if unset [env: NAP_DIR=]
  -u, --repository <REPOSITORY>  Repository name
  -n, --name <NAME>              Human-readable name
  -v, --verbose                  Enable verbose debug logging
  -a, --author <AUTHOR>          Author identifier [default: nap-cli]
  -h, --help                     Print help
