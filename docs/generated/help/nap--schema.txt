Print a JSON Schema for manifest or commit types

Usage: nap schema [OPTIONS] <NAME>

Arguments:
  <NAME>  Schema name: 'manifest' or 'commit'

Options:
  -d, --base-dir <BASE_DIR>  Base directory for universe repositories. Defaults to $NAP_DIR, or ~/.nap if unset [env: NAP_DIR=]
  -f, --format <FORMAT>      Output format: json, yaml [default: json]
  -v, --verbose              Enable verbose debug logging
  -h, --help                 Print help
